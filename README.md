 Unit Converter (Light Mode Version)

This is a clean, user-friendly web application designed for instant **Metric/Imperial** unit conversions. This "Light Mode" edition features a bright, warm aesthetic optimized for high visibility and ease of use.

🚀 Features

* **Real-time Conversion**: Converts length, volume, and mass simultaneously with a single click.
* **Precision Accuracy**: All results are calculated and rounded to **2 decimal places**.
* **User Validation**: Includes a safeguard to alert users if they attempt to convert without entering a number.
* **Clean Interface**: Custom CSS removes standard browser input "spinners" for a more streamlined look.

🛠️ Conversion Formulas

The application performs calculations based on the following standards:
* **Length**: 1 Meter $\approx$ 3.281 Feet
* **Volume**: 1 Liter $\approx$ 0.264 Gallons
* **Mass**: 1 Kilogram $\approx$ 2.204 Pounds

🎨 Design & Aesthetic

The **Light Mode** version uses a warm, inviting color palette:
* **Primary Header**: A rich amber (`#B45309`) background with `whitesmoke` text.
* **Body Content**: A soft, light cream (`#FEF3C7`) background for the results section.
* **Result Cards**: Floating `whitesmoke` cards with high-contrast text to separate different unit categories.
* **Responsive Layout**: Built using **CSS Flexbox** and **Grid** to ensure the converter is centered and scales well.

📂 Project Structure

* `index.html`: The structural foundation of the converter.
* `index.css`: The styling layer that defines the light-mode theme.
* `index.js`: The logic engine that handles the mathematical conversions and DOM updates.
